VRaptor 4 delivers high productivity to your Java Web applications on top of [CDI](http://cdi-spec.org/). VRaptor is an opensource MVC framework with a large developers and users community.

Download Looking for VRaptor 3?
## Really simple!
This is your first logic on `FirstController.java`: ~~~ #!java @Controller public class FirstController { @Inject private Result result; @Get("/home") public void home() { result.include("message", "Hello, VRaptor 4!"); } } ~~~
And HTML file placed at `WEB-INF/jsp/first/home.jsp`: ~~~ #!jsp Home page ${message} ~~~
It is done! Now you may just access [localhost:8080/home](http://localhost:8080/home). See more examples [on our documentation](/en/docs/one-minute-guide).
