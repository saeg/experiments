O VRaptor 4 traz alta produtividade para um desenvolvimento Java Web rápido e fácil com [CDI](http://cdi-spec.org/). É um framework MVC opensource com grande comunidade de desenvolvedores e usuários!

Download Procurando pelo VRaptor 3?
## Simples assim!
Sua lógica no `PrimeiroController.java`: ~~~ #!java @Controller public class PrimeiroController { @Inject private Result result; @Get("/home") public void home() { result.include("mensagem", "Olá, VRaptor 4!"); } } ~~~
E o HTML em `WEB-INF/jsp/primeiro/home.jsp`: ~~~ #!jsp Página inicial ${mensagem} ~~~
Pronto! Agora é só acessar [localhost:8080/home](http://localhost:8080/home). Veja mais exemplos [em nossa documentação](/pt/docs/guia-de-1-minuto).

Por que usar o VRaptor

    Usar o VRaptor 4 é simples e intuitivo. Você atingirá níveis altíssimos de produtividade com Java para Web.

    Em pouco tempo você conseguirá aprender tudo o que é necessário para desenvolver suas aplicações com o VRaptor.

  • Testabilidade

    Escreva código modularizado e desacoplado do VRaptor. Sua aplicação fica altamente testável e de fácil manutenção.

  • Economia

    Economize muitas horas de trabalho com a alta produtividade do VRaptor, a facilidade em treinar a sua equipe e a qualidade final do seu projeto.
